python-multi-game-suite
Interactive Python application with three text-based games: Word Scrabble, Word Guesser, Password Finder

Project Overview
An interactive Python application featuring three text-based games in one interface. Developed as a group project during 3rd semester at University of Central Punjab.

Games Included
1. **Word Scrabble** - Score-based word game with dictionary validation
2. **Word Guesser** - Hint-based word guessing with multiple difficulty levels
3. **Password Finder** - Simulation demonstrating brute-force attack principles

Technologies Used
- Python 3.x
- File I/O Operations
- Modular Programming
- Random Module
- Time Module
